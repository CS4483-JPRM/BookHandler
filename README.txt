How to run:

Game > TheBookHandler.exe 


Features:
- Basic Main menu
- Pause screen to quit 
- Customer Spawning + navigation through items
- GUI including money, time and days
- Building mode to drag new objects onto the scene
- Added model animations to the customers
- Added Day, night light changes
- Customers buying stuff gives players money to buy more stuff
- Having more bookshelves -> more customers can shop at a time
- Used models to build the environment/scene

Assets used:
The models and animations used come from stuff developed by Synty Studios
